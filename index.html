inspired by the brilliant Kevin Powell and cube css

a collection of css styles built to be lightweight and customizable

flex-row

A wrapper that applies display: flex; with a default gap of 1rem. The var(--gap) is included to allow the gap to be overwritten.

grid-flow

A similar wrapper to flex-row that instead sets display: grid;

Buttons

Buttons are minimal elements that only include style rules for background & text color.

Cards

Cards are predefined containers that include several optional child elements. The maximum width of a card is set to 30rem, and card-images are by default set to 480 width.
